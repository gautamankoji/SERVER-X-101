##### Problem Statement

Given 2 numbers A, B. Print A+B

**Input:**

* 1st line contains A
* 2nd line contains B

**Output:**

Print A+B

**Sample Input:**

{{< file "terminal" "Sample Input" >}}

```md
1 2
```

**Sample Output:**

{{< file "terminal" "Sample Ouput" >}}

```md
3
```

## Solution

<!-- **Approach:** -->

{{< file "python" "count_of_family_members.py" >}}

```py
a, b = map(int, input().split())
print(a + b)
```
